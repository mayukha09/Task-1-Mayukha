# Raw Sales Dataset

## Overview

This dataset contains sales transaction records used for data analysis, SQL querying, and data visualization projects. The data serves as the foundation for extracting business insights and creating reports.

## Dataset Description

The dataset includes information related to:

* Order ID
* Product Name
* Quantity
* Unit Price
* Total Price
* Order Status
* Customer Details
* Order Date

## Data Cleaning

Before analysis, the dataset was reviewed for missing values and inconsistencies.

Actions performed:

* Blank cells were replaced with "None".
* Data format consistency was checked.
* Records were validated for analysis purposes.

## Usage

This dataset was used in:

* Exploratory Data Analysis (EDA)
* SQL Data Analysis
* Data Visualization Projects

## Files Included

* raw_data.csv.xlsx

## Purpose

The purpose of this dataset is to practice data cleaning, analysis, querying, and visualization techniques using real-world business data.

## Conclusion

The dataset provides a structured source of sales information that can be used to generate insights, create visualizations, and support data-driven decision-making.
